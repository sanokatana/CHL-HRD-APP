## Installation

- Clone the repository
- Copy .env.example to .env
- Set the DB_ environment variables in .env file
- Create a database with the name specified in DB_DATABASE
- ```composer install```
- ```npm install```
- ```php artisan key:generate```
- ```php artisan migrate:fresh --seed```
- ```php artisan serve```
- ```npm run dev```
